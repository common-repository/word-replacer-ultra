== Description ==

Word Replacer Ultra has the capability to search and replace any word which is on the WordPress website. which will increase the efficiency of the work by assisting the developers in searching a word from a vast database of WordPress even though if it is in any post, custom post, page and product, and replacing it from the word developer wants. Word Replacer Ultra is an effective tool that can easily be download and installed.

### Plugin Features

* Search any word from the WordPress website
* Replace any word on the WordPress website
* Replace a word in posts
* Replace a word on the page
* Replace a word in custom post types

== Installation ==

* Upload to the /wp-content/plugins/ directory. Or Directly upload from your Plugin management page.
* Activate the plugin through the ‘Plugins’ menu in WordPress
* In the admin menu you will see Word Replacer Ultra

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.6 or greater
